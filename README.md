HELLO HTTP
===

## API

### GET /

If there are no queries this will return the response text `hello stranger`. There are 2 queries salutation and name, the defaults are `salutation=hello` and `name=stranger`. You can query `salutaion=hi` and `name=bob` and it will return `hi bob`.

### GET /fact

This will return a random fact about http.
